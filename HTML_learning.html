HTML element

the HTML element is everything from the start tag to the end tag

elements can be nested, such as the body tag nested in the html tag

emply tag is the tag without any content, such as br which defines a line break

tags are not case-sensitive, the HTML5 standards doesn't require lowcase tags, but W3C recommends lowercase in HTML, and demands lowercase in XHTML

HTML attributes

  • HTML links are defined with the "href" attribute, which specifies the link address

  • HTM image are defined with the "src" attribute, which specified teh filename of the image source

    images have a set of attributes, which specifies the width and height of the image, such as the width and height attributes

    images have the alt attribute, which specifies the alternative text if the image cannot be displayed

  • style attribute is used to specify the styling of an element, like color, font, size, etc

  • the language of a document can be declared in the html tag with the "lang" attribute, it's important for accessibility and search engine

    For example, the "en-US", the first two letters specify the language, if there is a dialect, use two more letters

  • the title attribute is added to P tag or any other tag, the value of this attribute will be displayed as a tooltip(文本提示) when you mouse over the paragraph

HTML headings

we have heading tags from h1 to h6

it's important to use headings, search engines use these to index the structure and content of your web page

"hr" tag defines a thematic break in a HTML page, is displayed as a horizontal rule

HTML paragraphs

HTML display: with HTML, you cannot change the output by adding extra spaces or extra lines in your HTML code

The browser will remove any extra spaces and extra lines when the page is displayed

"br" tag defines a line break, we can use it within a paragraph without starting a new paragraph, as below

  • first condition

    list1

    list2

  • second condition

    list1
    list2

"pre" element can define preformatted text, so that any extra spaces or lines will be preserved

For example

        this is the test case one with 3 indentation
            this is the test case two

HTML styles

setting the style of an HTML element, can be done with the style attribute

the HTML style attribute has the following syntax

tagname style='property:value;'

HTML formatting

    formatting elements are designed to display special types of text:
  • bold text with b
  • important text with strong
  • italic text with i
  • emphsized text with em
  • marked text with mark
  • small text with small
  • deleted text with del
  • inserted text with ins
  • subscript(下标) text with sub
  • superscript(上标) text with sup
